## 포트폴리오 사이트 만들기 프로젝트

## 완성작 보기

## 사용 스택

1. react(https://ko.react.dev/) 를 사용하여 사이트를 번들링하고 관리합니다.

2. gsap(https://greensock.com/gsap) 를 이용하여 패럴랙스 효과를 줍니다.

3. lenis(https://www.lenis.dev/) 를 이용하여 스므스 효과를 구현합니다.

4. netlify(https://www.netlify.com/) 를 통해 사이트를 배포합니다.

5. git(https://github.com/) 을 사용하여 파일을 관리합니다.

HTML, CSS 기반으로 웹사이트의 기본 레이아웃 설계하고, 웹 표준 및 웹 접근성을 준수하여 작업합니다.

ARIA(Accessible Rich Internet Applications)

## 프로젝트 실행

- react를 설치합니다. npm create vite@latest 타이틀

- react-router-dom을 설치합니다. npm install react-router-dom

- sass를 설치합니다. npm install sass

- gsap를 설치합니다. npm install gsap

- lenis를 설치합니다. npm instal lenis
